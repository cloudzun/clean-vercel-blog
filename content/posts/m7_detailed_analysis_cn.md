### 3.2 叙事验证 (Narrative Check)

#### 为什么 GOOG 领跑而 MSFT 落后？

**数据现象**:
- 两者都是云计算龙头
- 都在 AI 上投入巨资
- 但 GOOG 1年 +65%，MSFT 1年 -1%

**新闻叙事**:
- GOOG: Gemini 获得市场认可，AI 战略清晰
- MSFT: OpenAI 关系不明确，Azure ROI 有疑问

**逻辑串联**:
虽然财务数据相近，但市场对 **AI 商业化路径的确定性** 定价不同。Google 的 Gemini 已被广泛接受，而 Microsoft 的 Copilot 商业化进度不及预期。

**结论**: 这不是基本面差异，而是 **预期差** 导致的估值分化。

### 3.3 风险折价模型 (Risk Discount)

#### AAPL 的监管风险

**负面新闻**: DOJ 反垄断诉讼
**影响机制**:
- 短期: 估值倍数压制 (Multiple Compression)
- 中期: 商业模式改变风险 (如被迫开放侧载)
- 长期: 利润率下降可能

**定量影响**:
- 正常情况: P/E 35-40x
- 监管风险下: P/E 30-33x (当前水平)
- **建议**: 下调目标价 5-10%，直到诉讼有明确结果

## 第四部分：M7 特异性分析

### 4.1 各家的关键监控指标

#### NVIDIA - 需求确认
**盯紧**: 下游 Capex 新闻
- 如果 Meta/Google/Tesla 的 GPU 订单减少 → 股价暴跌
- 如果 Capex 新闻持续增强 → 估值还有上升空间

#### Microsoft - 预期修复
**盯紧**: Azure 增速是否稳定在 25% 以上
- 如果 Azure 增速 > 30% → 股价反弹到 $450-480
- 如果 Azure 增速 < 20% → 继续下跌到 $370-390

#### Amazon - 利润率改善
**盯紧**: AWS 营业利润是否逐季改善
- 如果 AWS 利润率 > 35% → 股价反弹
- 如果 AWS 利润率 < 30% → 继续承压

#### Apple - 机构持仓
**盯紧**: 机构持仓比例是否下降
- 如果机构持仓下降 > 2% → 大资金在撤退
- 如果回购力度加大 → EPS 支撑

#### Meta - Capex 指引
**盯紧**: 资本开支是否导致利润率预警
- 如果 Capex 占收入 > 15% 且 ROI 不明确 → 股价下跌
- 如果 AI 收入开始贡献 → 股价反弹

#### Tesla - 毛利率支撑位
**盯紧**: Gross Margin 是否跌破 15%
- 如果毛利率 < 15% → 技术面破位，可能继续下跌
- 如果毛利率稳定在 17-18% → 形成底部

#### Google - 搜索市场份额
**盯紧**: 搜索广告收入增速
- 如果搜索增速 > 15% → 继续领跑
- 如果搜索增速 < 10% → 增长乏力
